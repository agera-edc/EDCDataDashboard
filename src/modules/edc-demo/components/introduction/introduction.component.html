The Eclipse Dataspace Connector

The Eclipse Dataspace Connector provides a framework for sovereign, inter-organizational data exchange. It will implement the International Data Spaces standard (IDS) as well as relevant protocols associated with GAIA-X. The connector is designed in an extensible way in order to support alternative protocols and integrate in various ecosystems.

EDC Demo (this application)

This EDC demo shows (some) features of the Eclipse Dataspace Connector. It consists of this application, built with Angular, and an Eclipse Dataspace Connector extension, that provides the backend API for this frontend. This application is not intended for production usage and is 'work in progress'. Some of the features are faked to some degree, or some assumptions are made, to close the gap where features are in the flux on the connector side. The intention is to show the general flow (where possible and useful).

Write your first extension

Example use cases, that you can try out with this application, are:

  • View the asset catalog of your Dataspace using the Catalog Browser
  • Search for assets in your Dataspace using the Catalog Browser
  • View your existing contracts using the Contract Viewer
  • Transfer an asset in your Dataspace using the Catalog Browser
  • Create a policy and apply it to assets in your Dataspace using the Contract Definition Editor
  • View which assets have been transfered in your Dataspace using the Transfer History Viewer
  • Publish a new asset into your Dataspace using the Asset Publisher
